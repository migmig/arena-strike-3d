# 3D Shooting Game — PRD

## 1. 개요

| 항목 | 내용 |
|---|---|
| 제품명 | (가칭) Arena Strike 3D |
| 장르 | 1인칭 / 3인칭 아레나 슈팅 (싱글플레이어, 웨이브 기반) |
| 플랫폼 | 웹 브라우저 (데스크톱 우선, Chrome/Edge/Safari 최신) |
| 기술 스택 | TypeScript, Three.js, Vite, Rapier(물리), Howler(오디오) |
| 타겟 유저 | 캐주얼 ~ 미드코어 FPS 유저, 빠른 1세션(5~15분) 선호 |
| 핵심 가치 | "브라우저 열고 30초 내 사격 시작" — 설치 없이 즐기는 본격 3D 슈팅 |

## 2. 목표 & 비목표

### 목표 (MVP)
- 단일 맵에서 웨이브 기반 적 처치 루프 완성
- 무기 3종, 적 3종, 5웨이브 보스전 1회
- 60fps @ 1080p (M1 / GTX 1060 기준)
- 첫 진입에서 튜토리얼 없이 조작 학습 가능

### 비목표 (MVP 제외)
- 멀티플레이어 / 네트워크 동기화
- 모바일 터치 컨트롤
- 인앱 결제 / 계정 시스템
- 광범위한 커스터마이징(스킨, 무기 모드 등)

## 3. 핵심 게임플레이 루프

1. **준비** — 메인 메뉴에서 난이도 선택 → 맵 로드
2. **전투** — 웨이브 시작, 적 스폰 → 처치 → 픽업(탄약/체력)
3. **보상** — 웨이브 클리어 시 무기/퍽 1개 선택 (3택 1)
4. **반복** — 다음 웨이브, 점진적 난이도 상승
5. **결과** — 사망 또는 보스 클리어 시 점수/시간 표시 → 재도전

목표 세션 길이: **8~12분**

### 페일 스테이트 (Fail State)
- HP 0 도달 시 즉시 사망 → 슬로우모션 2초 + 결과 화면
- 부활 없음 (로그라이크 1-life 정책으로 긴장감 유지)
- 보스 도중 사망 시 보스 처음부터 재시작 (체크포인트 없음)
- 결과 화면: 점수, 처치 수, 정확도, 최장 콤보, 도달 웨이브 → 재도전 / 메뉴

### 난이도 옵션
| 난이도 | 적 HP | 적 데미지 | 적 수 | 픽업 빈도 |
|---|---|---|---|---|
| Easy | 0.75× | 0.75× | 0.8× | 1.25× |
| Normal | 1.0× | 1.0× | 1.0× | 1.0× |
| Hard | 1.25× | 1.25× | 1.2× | 0.85× |
| Nightmare | 1.5× | 1.5× | 1.5× | 0.7× |

### 온보딩 (튜토리얼 없는 학습)
- 첫 진입 시 빈 아레나, 화면 중앙에 컨트롤 힌트 페이드인 (3초)
- 첫 더미 적 등장 시 사격 힌트 표시
- 첫 재장전 트리거 시 "R" 키 힌트
- 모든 힌트는 1회만 노출 (LocalStorage 플래그)

## 4. 컨트롤 & 카메라

| 입력 | 동작 |
|---|---|
| W/A/S/D | 이동 |
| Space | 점프 |
| Shift | 대시 (쿨다운 3초) |
| Mouse | 시점 |
| LMB | 사격 |
| RMB | 조준(ADS) |
| R | 재장전 |
| 1/2/3 | 무기 전환 |
| Q | 무기 빠른 교체 (이전 무기) |
| F | 픽업 상호작용 (선택형) |
| V | 카메라 토글 (1인칭 ↔ 3인칭) |
| Esc | 일시정지 |

- 마우스 감도, FOV(70~110), 인버트 Y 옵션 제공
- 모든 키 리바인딩 가능 (액션 매핑)
- Pointer Lock API 사용

### ADS (조준) 명세
- RMB 홀드 시 FOV가 0.15초간 70 → 50으로 보간
- 이동속도 0.6× 감소, 탄퍼짐(spread) 0.4× 감소
- ADS 중 감도 0.7× 자동 적용

### 카메라 모드
- **1인칭(기본)**: 플레이어 눈 위치 (1.6m), 무기 viewmodel 표시
- **3인칭**: 어깨 너머 오프셋 (Vector3(0.4, 0.3, -2.0)), 캐릭터 메시 가시
- 토글 시 0.3초 보간 (Cinematic transition)
- 멀티플레이어/PvP 밸런스 영향 없음 (싱글 전용)

## 6. 적 (MVP)

| # | 적 | HP | 행동 | 보상 |
|---|---|---|---|---|
| 1 | Grunt | 50 | 직선 추적, 근접 공격 | 탄약 |
| 2 | Shooter | 80 | 거리 유지 + 사격 | 탄약/체력 |
| 3 | Charger | 150 | 돌진 후 폭발 | 체력 |
| B | Boss | 1500 | 3페이즈, 패턴 사격 | 클리어 |

스폰 규칙: 웨이브마다 적 수 +30%, HP +10%

### 적 AI 감지
- 시야각: 120°, 시야 거리 25m (벽 occlusion 반영)
- 청각 반응: 발사 시 15m 내 적 어그로 갱신
- 어그로 우선순위: (1) 최근 데미지 가한 플레이어 → (2) 최단 거리

### 보스 페이즈 명세
| 페이즈 | HP 구간 | 행동 패턴 |
|---|---|---|
| P1 | 100% ~ 67% | 단발 사격 + 좌우 회피 이동 |
| P2 | 67% ~ 33% | 산탄 3-way + 미니언 2기 소환 (10초 쿨) |
| P3 | 33% ~ 0% | 돌진 + 광역 폭발 (반경 5m), 페이즈 전이 시 2초 무적 |

목표 처치 시간: **90~120초** (Normal 기준)

### 밸런싱 타겟
- 평균 TTK (Grunt): Pistol 3발, SMG 5발, Shotgun 1발 (근접)
- 웨이브당 소요 시간: 60~90초
- 플레이어 평균 DPS (퍽 0개 기준): ~80

## 7. 맵 & 레벨

- **MVP 맵 1개**: 30m × 30m 아레나, 엄폐물 6~8개, 2층 구조
- 픽업 스폰 포인트 8곳 (탄약 4, 체력 2, 파워업 2)
- 환경 테마: 산업 폐허(콘크리트 + 네온)

### 픽업 명세
| 종류 | 효과 | 재스폰 |
|---|---|---|
| 탄약팩 (소) | 활성 무기 탄창 +50% | 20초 |
| 탄약팩 (대) | 모든 무기 탄창 가득 | 45초 |
| 체력팩 | HP +40 (오버힐 불가) | 30초 |
| 파워업: Damage Boost | 10초간 데미지 ×2 | 60초 |
| 파워업: Quad Speed | 8초간 이동속도 +50% | 60초 |

- 픽업은 자동 획득 (충돌 시), 파워업만 F키 상호작용
- 픽업 위치는 네온 빔으로 시각 표시
- 8개 스폰 노드 = 탄약(소) 2 + 탄약(대) 2 + 체력 2 + 파워업 2 (고정 배치)
- 웨이브 클리어 보상: HP +30 자동 회복 (오버힐 불가)

## 8. 메타 시스템

### 퍽 시스템 (웨이브 클리어 보상)
- 이동속도 +10%
- 재장전 속도 +25%
- 치명타 확률 +15%
- 최대 체력 +25
- 대시 쿨다운 -1초
- (총 12종 중 무작위 3택 1)

### 점수 & 콤보
`score = (적 처치 점수) × (콤보 배율) + (시간 보너스)`

| 적 | 기본 점수 |
|---|---|
| Grunt | 100 |
| Shooter | 150 |
| Charger | 250 |
| Boss | 5000 |

**콤보 시스템**:
- 처치 간 간격 ≤ 3초 유지 시 콤보 카운트 +1
- 배율: 1× → 1.2× (3 콤보) → 1.5× (7) → 2.0× (15) → 2.5× (30+)
- 피격 시 콤보 -50%, 4초 동안 처치 없으면 0으로 리셋
- 헤드샷 = 2 콤보 카운트로 인정

### 데미지 피드백
- 히트마커: 적중 시 크로스헤어 중앙 X 표시 (0.1초)
- 크리티컬: 노란색 히트마커 + 별도 SFX
- 데미지 넘버: 적 위치 위 3D 부유 텍스트, 크리는 크기 1.5×
- 피격 시: 화면 외곽 적색 비네팅 + 카메라 흔들림 (0.15초)
- 저체력(HP < 30): 화면 외곽 지속 적색 펄스

## 9. UI / UX

- **HUD**: 체력바(좌하), 탄약(우하), 미니맵(우상), 웨이브/콤보(상단 중앙)
- **메뉴**: 메인 → 시작 / 옵션 / 크레딧
- **결과 화면**: 점수, 처치 수, 정확도, 재도전 버튼
- 텍스트 최소화, 아이콘 우선

### 미니맵
- 우상단 200×200 px, 원형 마스크
- 플레이어 중심 회전형 (북쪽 고정 옵션 제공)
- 표시: 적(빨강), 픽업(녹/청), 보스(노랑)
- 시야 외 적은 외곽 화살표로 방향 표시

### 접근성
- 색맹 모드: Protanopia / Deuteranopia / Tritanopia 팔레트 (적/픽업 색상 대체)
- 자막: 모든 보스/적 음성 텍스트 표시
- 모션: 카메라 흔들림 OFF, 모션블러 OFF 토글
- 키 리매핑: 모든 액션 재할당 가능
- UI 스케일: 80% / 100% / 125% / 150%
- 색약 친화 크로스헤어 (도트/원/X자형)

### 일시정지 거동
- Esc → Pointer Lock 해제 + 게임 상태 동결 (물리/AI/타이머 모두 정지)
- 알트탭 시 자동 일시정지
- 윈도우 리사이즈 시 카메라 aspect 갱신, FOV 보정 옵션

## 10. 오디오

- 7일차 잔존율 ≥ 15%

## 15. Post-MVP 로드맵

- 멀티플레이어 코옵 (2~4인)
- 추가 맵 3종, 적 6종
- 무기 모드/스킨 시스템
- 모바일 / 게임패드 지원
- 리더보드, 일일 챌린지

## 16. 저장 & 데이터

- **저장 위치**: LocalStorage (5MB 한도 내)
- **저장 항목**:
  - 옵션 (감도, FOV, 키맵, 그래픽 프리셋, 오디오 볼륨)
  - 하이스코어 Top 10 (난이도별)
  - 튜토리얼 힌트 노출 플래그
  - 누적 통계 (총 처치, 총 플레이타임)
- **스키마 버전**: `{version: 1, ...}`, 마이그레이션 가능 구조
- **저장 없음**: 진행 중 세션 (1-life 정책)

## 17. 텔레메트리 (KPI 측정)

- **수집 이벤트**:
  - `game_start` (difficulty, timestamp)
  - `wave_complete` (wave_num, time_taken, deaths)
  - `player_death` (wave, weapon_used, total_kills)
  - `weapon_pickup` / `perk_choice`
  - `session_end` (duration, max_wave, final_score)
- **개인정보**: 익명 세션 ID만 수집, 개인 식별 정보 없음
- **동의**: 첫 진입 시 1회 옵트인 다이얼로그 (거부 시 비활성)
- **저장**: 자체 엔드포인트 (Cloudflare Workers + D1) 또는 PostHog
